# Track Plugin
Work time tracker.

##### Alias: track
##### Help: help track

## Commands
- ```add```: add project.
- ```stop```: stop timer.
- ```today```: show today time sheet.

## Usage
Add projects.
```cs
track add {ProjectName}
// example
track add TestProject
```
Start timer, type
```cs
track {ProjectName}
```
Stop timer
```cs
track stop
```
Get today sheet
```cs
track today
```
